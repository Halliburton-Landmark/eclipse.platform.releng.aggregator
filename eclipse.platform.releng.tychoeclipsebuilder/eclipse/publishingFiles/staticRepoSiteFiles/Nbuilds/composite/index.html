This software repository URL, http://download.eclipse.org/eclipse/updates/4.3-N-builds/ , provides access to a software repository for the Eclipse Project.

This particular repository site is for Kepler Nightly builds.

Do not mix with non-N-build repositories or installs!See Update Sites for more information.

N-builds are not recommeded for normal use, early testing, or build targets. They are primarily for committers to confirm a fix is in or compiles ok.

For more information about installing or updating software, see the Eclipse Platform Help.

There is also a collection of handy, downloadable all-in-one zip files available for many interests and platforms. Many people find these all-in-one packages the easiest way to get started.
